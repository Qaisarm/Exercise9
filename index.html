Bish-Bosh 2.0

 


Description:

Bish-Bosh lists all numbers between 1 and 100, but some exceptions: If the number is evenly divisible 3 listed 'Bish' instead of century. If the number is divisible by 4 lists ' Bosh ' instead the number. If the number is divisible by both 3 and 4 are written " Bish-Bosh " instead of the number.
