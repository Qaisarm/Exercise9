Bish-Bosh 2.0

Description:

Bish-Bosh lists all numbers between from 1 till the Loop value of your choice, There are some exceptions:
If the number is evenly divisible by Divisor One, it will print 'Bish' instead of Divisor One value you enter.
If the number is evenly divisible by Divisor Two, it will print 'Bosh' instead of Divisor Two value you enter.
If the number is evenly divisible by both Divisor One and Divisor Two it will print 'BishBosh' instead of that Number.
